# Spring Boot 3.x 快速入门
演示快速启动一个简单的Spring Boot工程项目。

## 开发环境
| 开发工具      | 版本号    |
| ------------- |--------|
| Java          | 21     |
| Maven         | 3.9.9  |
| Spring Boot   | 3.5.0  |
| IntelliJ IDEA | 2025.1 |

## 启动应用
执行QuickStartApplication.java主程序，访问路径
| URL样例地址                     | 描述                          |
| ------------------------------- | ----------------------------- |
| http://localhost/               | 返回欢迎信息              |

```
curl http://localhost/
```
